# Regards D'artisans

Regards D'artisans is a website about Arts and crafts.

## Technologies
We are using ``` NextJS ```, ``` React ```, ``` SCSS ```, ``` Vercel ``` and ``` THREE JS ```.

## Setup
We are using ```Yarn ```, in the app folder : 
```bash
yarn install
yarn local
```
